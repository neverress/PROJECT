image_dimensions: tuple[int, ...] = ¶

tuple: Размеры изображений.

Содержит три значения: - Высота изображения. - Ширина изображения. - Число цветовых каналов (3 для RGB).

label_file: str = ¶

str: Имя файла с метками классов.

Файл должен содержать два столбца: - ClassId: числовой идентификатор класса. - Name: название класса.

path: str = ¶

str: Путь к папке с изображениями для обучения и тестирования.

Эта папка должна содержать изображения, отсортированные по классам в отдельных подкаталогах. Каждый подкаталог представляет один класс.

test_ratio: float = ¶

float: Доля данных для тестового набора.

Определяет, какая часть данных будет выделена для тестирования. Например, значение 0.2 означает, что 20% всех данных будут использоваться для тестирования.

validation_ratio: float = ¶

float: Доля данных для валидационного набора.

Определяет, какая часть данных из тренировочного набора будет выделена для валидации. Например, значение 0.2 означает, что 20% тренировочных данных будут использованы для валидации.
