labels_path: str = ¶

str: Путь к файлу с названиями классов.

Файл должен быть в формате CSV и содержать два столбца: - ClassId: идентификатор класса. - Name: название класса. Этот файл используется для отображения названия класса, соответствующего предсказанию модели.

threshold: float = ¶

float: Порог уверенности модели для отображения результата.

Определяет минимальную вероятность (значение от 0 до 1), при которой модель считается уверенной в своём предсказании. Если вероятность предсказания ниже этого порога, результат не отображается.
